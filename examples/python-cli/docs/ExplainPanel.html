Pulls articles from NewsAPI for one or more topics, deduplicates by sha256, and merges tags when an article matches multiple topics.

2from typing import Iterable

4def run(self, topics: Iterable[str]):
5    seen: dict[str, Article] = {}
6    for topic in topics:
7        for article in self._fetch_topic(topic):

10            ).hexdigest()
11            if digest in seen:
12                seen[digest].tags.add(topic)
13            else:
14                article.tags = {topic}
15                seen[digest] = article

17def _fetch_topic(self, topic: str):
18    params = {"q": topic, "pageSize": 100}
19    if self.since:

22        )
23    return self.client.get("/everything", params=params)
  • L5Dedup by sha256(title+source) — re-ingesting the same article produces the same id, upsert is idempotent.
  • L10Tag fusion: an article retrieved via both Python and AI/ML topics inherits both tags.
  • L18page_size=100 is the NewsAPI free-tier maximum.
  • L21Truncate dates to YYYY-MM-DD — NewsAPI rejects timezone-aware values.

Single shared HNSW collection configured with cosine similarity — the architectural decision that drives every downstream distance computation.

13            "hnsw:search_ef": 64,

  • L1lru_cache(maxsize=1): one HTTP client for the whole process lifetime.
  • L11hnsw:space='cosine' — without this, ChromaDB defaults to L2 (Euclidean), which behaves differently for variable-length text.
  • L15HNSW is a logarithmic-time graph index — fast lookup across millions of vectors.
