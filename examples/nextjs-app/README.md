# Example: Next.js full-stack app

A small Next.js 14+ App Router app with server actions, demonstrating a frontend-primary documentation surface.

## Files

- [`docs/pipeline-map.json`](docs/pipeline-map.json) — produced by `/explore-pipeline`.
- [`components/ExplainPanel.tsx`](components/ExplainPanel.tsx) — generated by `/explain-panel`.

## Highlights

- Single-package repo (`roots: ["."]`).
- TypeScript/TSX snippets.
- 3 groups: routing / data / mutations.
- Brand color override on the "mutations" group — demonstrates the custom color object form.
